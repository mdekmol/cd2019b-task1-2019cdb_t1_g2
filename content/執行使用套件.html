執行使用套件

套件名稱 介紹 功能
V-rep

VREP集成了建模/编程/物理引擎。具有四款物理引擎，Bullet/ODE/Vortex/Newton。

其開源、快速、靈活的建模和模擬，同時有一定的模擬精度，為研發中非常重要的一部分。

特色:

  1. 建模和编程環境方便。

    2. VREP本身是開源的。使用LUA语言进行编程。

(LUA是一個體積很小的编程语言，但集成了C語言，Python，Matlab等等其他主流语言，使用上相對容易)

1.跨平台

2.六種編成方法

3.多種編成語言

4.完整的運動學計算器

5.路徑規劃

6.與現實接近的傳感器

(在檢測區域中的最短距離計算)
OnShape

Onshape 只要打開瀏覽器即可使用，使用的永遠都是最新版本，因此兼容性不會成為問题。

Onshape 具備了有限的實體建模能力，其幾何實體是由 Part Studio 這個多部件幾何體開發環境來生成的，没有文件的概念，因此也没有外部参照的依赖性问题。

特色:

1.線上協同設計。

2.容易上手的建模策略3.設計流程的重視。

繪圖功能其他繪圖軟體並無太大差異，其不同點在於:

1. 比較指令

2. 跟隨模式

3. 匯入/導入DXF檔案

4. 捷徑工具列

5. 加入評論

6. 插入時的快照模式
Github

GitHub是透過Git進行版本控制的軟體原始碼代管服務平台，由GitHub公司的開發者使用Ruby on Rails編寫而成。

特色:

1.提供付費帳戶和免費帳戶。

2.GitHub也提供了圖表功能。

3.Git 程式碼擁有開放和公開散播能力極佳的特性，提供了社群互動功能。

GitHub支援以下格式和功能：

1.文件：包括自動生成的、採用類Markdown語言的Readme檔案。

2.問題追蹤系統（同時可用於功能需求）

3.Wiki

4.GitHub Pages支援用戶透過軟體倉庫建立靜態網站或靜態部落格。

5.任務列表

6.甘特圖
CMSimfly

是一套簡單的網際內容管理系統, 採用 Flask 網際框架, 以 Beautifulsoup 解讀分頁內容.

以 Python 3 編寫, 利用 Flask 網路框架可以在伺服器運行, 讓使用者利用瀏覽器擷取動態 WWW 伺服器所送至近端的 html 加上 Javascript 程式碼, 然後在瀏覽器中顯示網際內容編輯器。		 1.建立可攜環境後即可多人協同
